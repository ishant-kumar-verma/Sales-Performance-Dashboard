# 📊 Sales Performance Dashboard | Power BI

## 📌 Project Overview

The **Sales Performance Dashboard** is an interactive Business Intelligence solution developed using **Microsoft Power BI**. The dashboard provides a comprehensive analysis of sales performance across products, customers, countries, and time periods.

It enables users to monitor key business metrics, identify sales trends, evaluate product performance, and gain actionable insights for data-driven decision-making.

This project demonstrates the complete Power BI workflow, including **data cleaning, transformation, data modeling, DAX calculations, and interactive dashboard design**.

---

## 🎯 Project Objectives

- Analyze overall sales performance
- Monitor key business KPIs
- Identify top-selling products
- Compare sales across different countries
- Track monthly sales trends
- Analyze customer demographics
- Build an interactive dashboard for business users

## 🗂 Dataset Information

The project contains sales transaction data including:

- Product
- Customer Information
- Country
- Sales Amount
- Order Date
- Customer Age

The dataset was cleaned and transformed using **Power Query Editor** before building the data model.

---

## 🏗 Data Model

A **Star Schema** was implemented to ensure efficient filtering and high-performance reporting.

### Fact Table

- FactSales

### Dimension Tables

- DimCustomer
- DimProduct
- Date Table

Relationships were created using one-to-many cardinality following Power BI best practices.
